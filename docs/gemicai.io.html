gemicai.io package¶

Submodules¶

gemicai.io.pickle module¶

IO module is used in order to interface with a file system. Entirety of this module is meant to be swapped out if superior tools or libraries arise in the future or more performance is needed.

gemicai.io.pickle.get_path_with_extension(file_path, obj)[source]¶

Returns file path with a correct extension. Dependent on the object type.

Parameters

  • file_path (str) – path to a file without it’s extension eg. /home/test/test_file. If correct extension is present path will be changed

  • obj (Union[gem.Classifier, gem.ClassifierNode]) – object which extension type we want to return

Returns

gemicai.io.pickle.load(file_path, zipped=False)[source]¶

This function is used to load back a Gemicai object from the file system.

Parameters

  • file_path (str) – valid path to a file

  • zipped (bool) – whenever the specified file is zipped

Returns

a valid Gemicai object

Raises

Exception – raised if object could not have been loaded in

gemicai.io.pickle.save(file_path, obj, zipped=False)[source]¶

This function saves a specified object in a given file.

Parameters

  • file_path (str) – valid path to a file. File itself does not have to exist.

  • obj (Union[gem.Classifier, gem.ClassifierNode]) – object to save

  • zipped (bool) – whenever the file should be zipped

Returns
